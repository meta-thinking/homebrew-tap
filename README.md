# heshamh96/homebrew-tap

```bash
brew tap heshamh96/tap
brew install --cask --no-quarantine ai-proxy
brew upgrade --cask ai-proxy     # updates
```

**ai-proxy** — Mac app that exposes your AI subscriptions (Gemini, Claude, …) as one
local OpenAI-compatible API for tools like n8n, opencode, Cursor, and anything else
that accepts a custom base URL. The app bundle contains no secrets.
